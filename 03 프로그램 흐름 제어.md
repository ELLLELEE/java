# 03 프로그램 흐름 제어
## - 컴퓨터와 대화
- scanner 이용
- 사용 방법
  - import java.util.Scanner;  -> Scanner 클래스 임포트
  - Scanner sc = new Scanner(Ststem.in);   -> 입력도구 선태, 스캐너 객체 생성
  - int lineNum = sc.nextInt();   -> 입력 받아서 변수에 저장
  - sc.close(); -> 스캐너 객체 닫기
- 데이터 타입력 입력 메서드
  - nextInt() -> 정수값
  - nextDouble() -> 실수값
  - next() -> 문자열
  - nextLine() -> 문자열(한줄전체)

## - 조건문
- if or switch
  - if
    - 조건식을 감싸는 괄호 필수
    - 중괄호 존재
#### * 문자열(string) 비교할때 .equals() 메서드 사용 해야함, == 사용 못함
  - switch
    - 조건이 명확한 값의 경우 : 정수형 값
    - 괄호안에 조건식이 들어가면 안됨
## - 반복문
- for , while , do-while
  - for
    - 기본 구조 (초기화식; 조건식; 증감식) -> for (int i = 0 ; i <10 ; i++)
    - 향상된 for 문 기본 구조 (변수 : 리스트) -> for (double value : list)
  - while
    - 조건이 만족하는 동안 계속 진행
    - while(rate > 0.1)
  - do-while
    - 조건이 만족하든 말든 일단 실행문 먼저 실행
